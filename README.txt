# Maribel y Cristian — PWA

## Cómo usar
1. Coloca este folder en tu computadora o celular.
2. Añade tu archivo de audio legalmente obtenido como `Married-Life-Up.mp3` junto al `index.html`.
3. Sirve la carpeta en un servidor local (necesario para PWA):
   - Con Python: `python -m http.server 8080` y abre http://localhost:8080/
   - O usa la extensión **Live Server** en VS Code.
4. Abre la URL en Chrome/Edge (móvil o desktop). Verás el botón "Instalar app".
5. Una vez instalada, funcionará offline (el audio requiere haber sido reproducido en la sesión).
